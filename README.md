# LifeLoop - Life Management App

A modern, responsive web application for organizing and managing your daily tasks, categories, and life events.

## Features

- **Modern Design**: Clean, contemporary interface with day/night mode
- **Task Management**: Create categories and add tasks with due dates
- **Calendar View**: Visual calendar to track tasks by date
- **Analytics**: Track your productivity and task completion
- **Responsive**: Works perfectly on desktop, tablet, and mobile
- **Chatbot Assistant**: Built-in AI assistant to help you navigate the app
- **Data Export**: Export your data for backup

## Getting Started

1. Open `index.html` in your web browser
2. Create an account or sign in
3. Start organizing your life!

## Troubleshooting

### If the app doesn't load properly:

1. **Clear browser cache**: Press Ctrl+F5 (or Cmd+Shift+R on Mac)
2. **Check console errors**: Press F12 and look for any error messages
3. **Reset app data**: Go to Settings → Reset App
4. **Try a different browser**: Chrome, Firefox, or Safari

### Common Issues:

- **Login not working**: Any email/password combination works for demo purposes
- **Tasks not saving**: Check if localStorage is enabled in your browser
- **Sidebar not working on mobile**: Tap the menu button (☰) in the top-left
- **Dark mode not working**: Click the sun/moon icon in the header

### Browser Compatibility

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+
